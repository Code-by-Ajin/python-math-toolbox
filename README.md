🧮 Python Math Toolbox v1.0

A comprehensive, lightweight, and professional console-based mathematical utility toolkit. Built entirely with Python's standard library, this application offers a suite of tools for number theory, geometry, statistics, and number system conversions, all wrapped in an easy-to-use Object-Oriented architecture.

---

## ✨ Features

Currently, the toolbox includes the following core functionalities, with an architecture designed to easily scale to 50+ features:

**🔢 Number Theory**
* Prime Number Checker
* Fibonacci Sequence Generator
* Factorial Calculator
* Armstrong Number Checker

**🔄 Number System Converter**
* Decimal to Binary, Octal, and Hexadecimal

**📐 Geometry Calculator**
* Area of a Circle
* Area of a Rectangle

**📊 Statistics Calculator**
* Calculates Mean, Median, Mode, Variance, Maximum, and Minimum from user-provided datasets.

---

## 🏗️ Architecture

The project has been streamlined into a single-file, **Object-Oriented Programming (OOP)** structure. This ensures the code is highly readable, easily distributable, and simple to expand. 

* `NumberTheory`: Static methods for core mathematical validations and sequences.
* `Converter`: Handles base-n number conversions.
* `Geometry`: Manages formulas for geometric shapes.
* `StatisticsCalc`: Processes arrays of data for statistical analysis.
* `MathToolboxApp`: The main application class that handles the console UI, screen clearing, and user input routing via a clean dictionary mapping system.

## 🚀 Getting Started

### Prerequisites
Because this project strictly utilizes Python's built-in standard libraries (`math`, `statistics`, `os`, `sys`, `random`), there is **no need** for external dependencies or a `requirements.txt` file. 

You only need **Python 3.6 or higher** installed on your system.

### Installation & Usage
1. Clone this repository or download the `python_math_toolbox.py` file.
2. Open your terminal or command prompt.
3. Navigate to the directory containing the file.
4. Run the program using:

```bash
python python_math_toolbox.py
